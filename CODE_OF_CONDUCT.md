# Code of Conduct

Context Ray contributors are expected to create a welcoming, respectful, and harassment-free environment.

## Expected behavior

- Be considerate, constructive, and specific.
- Assume good intent while addressing technical and interpersonal impact directly.
- Respect different backgrounds, identities, experience levels, and communication styles.
- Accept responsibility, apologize when appropriate, and repair harm.

## Unacceptable behavior

Harassment, discrimination, threats, doxxing, sexualized attention, sustained disruption, and publishing private information without permission are not acceptable in project spaces.

## Enforcement

Project maintainers may edit or remove contributions and temporarily or permanently restrict participation when behavior violates these expectations. Enforcement decisions should be proportionate, documented privately, and protect the safety and privacy of affected people.

Until a public moderation contact is configured, report conduct concerns privately to the repository owner through an agreed channel. This document is adapted in spirit from the Contributor Covenant 2.1; a future public release should add an explicit enforcement contact.
